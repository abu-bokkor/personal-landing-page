# Personal Landing Page

A modern, responsive personal landing page designed for performance and interactive user experience.

## Technologies Used
- **HTML5**: Semantic structure.
- **CSS3**: Responsive design with custom keyframe animations and Flexbox.
- **JavaScript (ES6+)**: Interactive particles and dynamic typewriter effect.

## Key Features
- **Responsive Design**: Optimized for mobile and desktop screens.
- **Interactive Background**: Custom particle animation.
- **Dynamic Text**: Smooth typing effects for a professional introduction.

## How to Run
1. Clone this repository:
   `git clone https://github.com/SEU-USUARIO/personal-landing-page.git`
2. Open `index.html` in your favorite web browser.
